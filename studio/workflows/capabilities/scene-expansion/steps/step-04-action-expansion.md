# Step 4: Action Expansion

## STEP GOAL:

Write detailed action prose for Build, Action, and Climax phases.

## MANDATORY RULES

- Explicit, not euphemistic
- Clinical terminology forbidden
- Sensory density throughout
- OBSERVER-CAMERA perspective

## MANDATORY SEQUENCE

### 1. Build Phase

Write approaching/initiation:
- Power dynamics establishing
- First physical contact
- Sensory details on contact

### 2. Action Phase

Write main activity:
- Explicit physical description
- Position changes
- Intensity escalation
- SFX integration points

### 3. Climax Phase

Write peak moment:
- Maximum intensity
- Physical resolution
- Sensory overload

### 4. Quality Check

```markdown
## Action Prose Check

- [ ] Explicit language used
- [ ] No euphemisms
- [ ] Sensory details present
- [ ] Escalation clear
```

### 5. Present MENU

```
"✅ Action expanded!

**Words:** ~{count}
**Phases covered:** Build, Action, Climax

**Chọn:** [C] Continue to Dialogue"
```
