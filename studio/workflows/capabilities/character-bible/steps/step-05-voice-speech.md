# Step 5: Voice & Speech

## STEP GOAL:

Create distinct voice and speech patterns for authentic dialogue.

## MANDATORY SEQUENCE

### 1. Speech Style

```
"**Voice & Speech**

{Name} nói chuyện như thế nào?

**Formality:**
- [ ] Formal / polite
- [ ] Casual / friendly
- [ ] Crude / vulgar
- [ ] Mix depending on context

**Speed:**
- [ ] Fast, excited
- [ ] Slow, deliberate
- [ ] Variable

**Quirks:**
- Hay dùng word/phrase gì?
- Có accent hoặc cách nói đặc biệt?
- Tic ngôn ngữ? (ví dụ: hay nói 'đó' cuối câu)"
```

### 2. R18 Dialogue Style

```
"**Khi trong scene R18:**

Họ nói như thế nào khi aroused?
- [ ] Trở nên crude/vulgar hơn
- [ ] Im lặng, chỉ moan
- [ ] Dirty talk master
- [ ] Begging/pleading

**Sample dirty talk của họ:**
- Khi dominant: '...'
- Khi submissive: '...'
- Khi climax: '...'"
```
