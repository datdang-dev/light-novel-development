**Anti-Hallucination**: 25 pts — Source/context contracts enforced.
**Fetish Targeting**: 20 pts — Fetish pack loaded, triggers defined.

## Critical Prompt Anti-Patterns
- Generic modifiers: "immersive", "vivid", "beautiful", "emotional"
- Safety/moral language: "respectful", "sensitive", "considerate"
- Missing constraints: "don't invent", "source-only", "1:1 mapping"
- Vague quality words: "better", "more detailed", "stronger"
- No output format definition

- **R18 Specifics**: Inject vulgar lexicon requirements, fetish triggers, anti-slop rules.
- **Token Brutal**: Cut any fluff. Use XML tags sparingly but precisely.
