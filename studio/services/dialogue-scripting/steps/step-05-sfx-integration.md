# Step 5: SFX Integration

## STEP GOAL:

Integrate sound effects to enhance dialogue and prose immersion.

## SFX GUIDELINES

```
PLACEMENT:
- Between dialogue lines
- During action descriptions
- At impact moments
- NOT every line (selective)

FORMAT:
- Romanized: *pan pan pan*
- Onomatopoeia: *zuchu... guchu guchu*
- Italicized in prose
```

## SFX CATEGORIES

### Impact Sounds
```
pan pan - skin slapping
pachin - light slap
bashi - heavy impact
```

### Wet Sounds
```
zuchu - insertion
nuchu - squelching
guchu guchu - sloppy/wet
picha picha - splashing
```

### Vocal Sounds
```
n~ - soft moan
a~ - surprised moan
haa haa - panting
hii - whimper
ngu~ - deep moan
```

### Climax Sounds
```
byuru byuru - release
doku doku - pulsing
biku biku - trembling
```

## MANDATORY SEQUENCE

### 1. Map SFX to Beats

```markdown
## SFX Integration

| Beat | Action | SFX | Placement |
|------|--------|-----|-----------|
| Build | insertion | *zuchu...* | after dialogue |
| Action | thrusting | *pan pan pan* | between lines |
| Climax | release | *byuru... doku doku* | during peak |
```

### 2. Integrate with Dialogue

```markdown
### Beat: Action

**{Char}:** "Ah... ah... nhanh hơn—"

*pan pan pan*

**{Char}:** "—nnngh! Sâu... sâu quá...!"

*guchu guchu*
```
