# Step 3: Psychological Profile

## STEP GOAL:

Create psychological depth through core wound, desires, and fears.

## MANDATORY SEQUENCE

### 1. Core Wound

```
"**Psychological Profile**

Mỗi character thú vị đều có 'core wound' - trauma/experience định hình họ.

**Core Wound là gì?**
Ví dụ:
- Bị bỏ rơi thuở nhỏ → tìm validation qua sex
- Bị kiểm soát quá mức → nổi loạn qua hành vi taboo
- Chưa bao giờ được chú ý → crave attention desperately

**{Name} có core wound gì?**"
```

### 2. Desires & Motivations

```
"**Họ MUỐN gì?**

Surface level: {what they say they want}
Deep level: {what they actually need}

Ví dụ:
- Surface: Muốn được yêu
- Deep: Muốn chứng minh mình đáng được yêu

**{Name} muốn gì ở 2 level?**"
```

### 3. Fears & Triggers

```
"**Họ SỢ gì?**

- **Sợ lớn nhất:** {biggest fear}
- **Trigger:** Điều gì khiến họ breakdown?
- **Defense mechanism:** Họ đối phó fear thế nào?"
```

### 4. Document Psychology

```markdown
## Psychological Profile

### Core Wound
{trauma/experience}

**How it manifests:**
- {behavior 1}
- {behavior 2}

### Desires
**Surface:** {stated want}
**Deep:** {actual need}

### Fears
**Primary Fear:** {biggest}
**Triggers:** {what sets them off}
**Defense:** {how they cope}
```

```

**Drives them to:** {behavior}
```
