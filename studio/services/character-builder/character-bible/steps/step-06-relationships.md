# Step 6: Relationships

## STEP GOAL:

Define relationship web with other characters in the story.

## MANDATORY SEQUENCE

### 1. Identify Connections

```
"**Relationships**

{Name} có relationships nào quan trọng?

List các character họ interact với và:
- Relationship type (lover, rival, prey, hunter, etc.)
- Power dynamic (who has power?)
- History (how they met/know each other)
- Tension (what creates drama?)"
```

### 2. Detail Each Relationship

For each connection:

```markdown
## Relationships

### {Name} ↔ {Other Character}

**Type:** {relationship type}
**Power Dynamic:** {who dominates}
**History:** {how they know each other}

**Public Behavior:** {how they act in front of others}
**Private Behavior:** {how they act alone}

**Tension Source:** {what creates conflict/drama}
**Sexual Dynamic:** {if applicable}
```

### 3. Relationship Patterns

```
"**Relationship Patterns**

{Name} có pattern nào trong relationships?
- Luôn là người dominant? Submissive?
- Có attachment style gì? (avoidant, anxious, secure)
- Có relationship they crave nhưng sợ?"
```

### 4. Present MENU

```
"✅ Relationships mapped!

**Key connections:** {count}
**Primary dynamic:** {pattern}

**Tiếp theo:** Finalize profile

**Chọn:** [C] Continue"
```
