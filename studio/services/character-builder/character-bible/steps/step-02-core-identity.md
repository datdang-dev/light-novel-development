# Step 2: Core Identity

## STEP GOAL:

Establish fundamental character identity: name, role, archetype, physical appearance.

## MANDATORY SEQUENCE

### 1. Name & Role

```
"**Core Identity**

Cho mình biết:

1. **Tên:** (có thể là bí danh)
2. **Vai trò:** protagonist / antagonist / supporting?
3. **Archetype:** (ví dụ: forbidden lover, manipulative gyaru, innocent kouhai)"
```

### 2. Physical Description

Guide detailed physical creation:

```
"**Physical Description**

Mô tả chi tiết:
- **Khuôn mặt:** Đặc điểm nổi bật
- **Tóc:** Màu, kiểu, độ dài
- **Thân hình:** Body type, đặc điểm
- **Trang phục thường:** Style họ hay mặc

Extra sexy:
- **Đặc điểm gợi cảm:** (ngực, mông, eo, etc.)
- **Điểm nhấn body:** Feature nào người ta notice first?"
```

### 3. Document Identity

```markdown
## Core Identity

**Name:** {name}
**Role:** {role}
**Archetype:** {type}

### Physical Description

**Face:** {description}
**Hair:** {color, style, length}
**Body:** {type, measurements if relevant}
**Usual Attire:** {style}

### Signature Features
- {feature 1}
- {feature 2}
```

### 4. Present MENU

```
"✅ Core identity established!

**{Name}** - {archetype}
{brief physical summary}

**Tiếp theo:** Psychological profile

**Chọn:** [C] Continue"
```
