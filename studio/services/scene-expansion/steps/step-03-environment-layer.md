# Step 3: Environment Layer

## STEP GOAL:

Write the environmental foundation with sensory grounding.

## MANDATORY SEQUENCE

### 1. Sensory Environment

Write environment prose including:
- Visual: lighting, colors, arrangement
- Smell: at least 1 environment smell
- Sound: ambient sounds
- Texture: surfaces, temperature

### 2. Output

Generate environment prose that:
- Sets atmosphere
- Grounds the scene physically
- Provides sensory foundation

### 3. Present MENU

```
"✅ Environment written!

**Sensory check:**
- Smells: {count}
- Sounds: {count}
- Textures: {count}

**Chọn:** [C] Continue to Action"
```
