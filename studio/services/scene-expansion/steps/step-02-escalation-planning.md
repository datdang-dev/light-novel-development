# Step 2: Escalation Planning

## STEP GOAL:

Structure the scene into proper escalation beats.

## ESCALATION TEMPLATE

```
1. SETUP (10-15%)
   - Establish setting
   - Character positions
   - Tension hint

2. BUILD (25-30%)
   - Approach/initiation
   - Rising tension
   - First contact

3. ACTION (35-40%)
   - Main activity
   - Peak physical engagement
   - Dialogue fragments

4. CLIMAX (10-15%)
   - Peak intensity
   - Resolution moment
   - Maximum sensory

5. AFTERMATH (10-15%)
   - Come-down
   - Physical residue
   - Emotional state
```

## MANDATORY SEQUENCE

### 1. Analyze Input Beats

Map user's concept to escalation phases:

```markdown
## Escalation Plan

| Phase | Content | Word Target |
|-------|---------|-------------|
| Setup | {what happens} | ~{X} words |
| Build | {what happens} | ~{X} words |
| Action | {what happens} | ~{X} words |
| Climax | {what happens} | ~{X} words |
| Aftermath | {what happens} | ~{X} words |
```

### 2. Present MENU

```
"✅ Escalation planned!

**Total phases:** 5
**Estimated length:** ~{X} words

**Chọn:** [C] Continue to Environment"
```
