# Step 5: Dialogue Integration

## STEP GOAL:

Weave dialogue and SFX into the action prose.

## MANDATORY SEQUENCE

### 1. Identify Dialogue Points

Mark where dialogue enhances:
- Power dynamics
- Emotional beats
- Intensity markers

### 2. Insert Dialogue

Match character voices:
- Use profile speech patterns
- Escalate with action
- Fragment at high intensity

### 3. Add SFX

At impact moments:
- Physical sounds
- Vocal reactions
- Wet/impact effects

### 4. Present MENU

```
"✅ Dialogue integrated!

**Lines added:** {count}
**SFX added:** {count}

**Chọn:** [C] Continue to Quality Check"
```
